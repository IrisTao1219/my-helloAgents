# 比特币价格显示应用 - 产品需求分析

## 1. 需求理解与分析

**核心价值**：为用户提供一个简洁、实时的比特币价格查询工具，帮助用户快速了解 BTC 当前市场状态。

**目标用户**：加密货币投资者、市场观察者、技术爱好者

**关键需求点**：
- 数据实时性：价格数据需要尽可能接近实时
- 数据准确性：使用可靠的数据源
- 交互友好：操作简单，信息一目了然
- 稳定可靠：网络异常时有合理的降级处理

## 2. 功能模块划分

| 模块 | 功能描述 | 优先级 |
|------|---------|--------|
| **数据获取模块** | 调用 API 获取 BTC 实时价格和 24h 变化数据 | P0 |
| **价格展示模块** | 主价格显示（大字体、USD 单位） | P0 |
| **趋势展示模块** | 24h 涨跌幅（%）+ 涨跌额（$），含颜色区分（涨绿跌红） | P0 |
| **刷新控制模块** | 手动刷新按钮 + 可选自动刷新 | P1 |
| **状态反馈模块** | 加载状态（spinner）+ 错误提示 + 最后更新时间 | P0 |

## 3. 技术选型建议

- **前端框架**：Streamlit（满足快速开发需求）
- **数据源**：推荐 **CoinGecko API**（免费、无需 API Key、稳定）
  - 备选：Binance API、CoinMarketCap API
  - 接口：`https://api.coingecko.com/api/v3/simple/price?ids=bitcoin&vs_currencies=usd&include_24hr_change=true`
- **HTTP 库**：`requests`
- **缓存机制**：`@st.cache_data(ttl=60)` 避免频繁请求
- **UI 组件**：
  - `st.metric()` 展示价格和涨跌（自带 delta 视觉效果）
  - `st.button()` 触发手动刷新
  - `st.spinner()` 显示加载状态
  - `st.error()` 显示异常信息

## 4. 实现优先级排序

**Phase 1（MVP，必须完成）**：
1. 接入 CoinGecko API 获取 BTC 价格
2. 展示当前价格 + 24h 涨跌幅/涨跌额
3. 加载状态与错误处理
4. 手动刷新按钮

**Phase 2（体验优化）**：
5. 显示最后更新时间
6. 涨跌颜色区分（红绿）
7. 页面标题、图标、布局美化

**Phase 3（可选增强）**：
8. 自动刷新功能（如每 30 秒）
9. 多币种支持

## 5. 验收标准定义

- [ ] 应用启动后能正确显示 BTC 当前 USD 价格
- [ ] 24h 涨跌幅（百分比）和涨跌额（美元）显示正确
- [ ] 点击刷新按钮能重新获取最新数据
- [ ] 涨跌颜色区分明确（涨绿跌红）

- [ ] API 请求失败时显示友好错误提示，不崩溃

- [ ] 页面布局清晰，价格信息醒目
- [ ] 显示数据最后更新时间

✅ **代码质量**：
